README: quart_training — учебный pet‑проект по освоению Quart
О проекте
quart_training — учебный пет‑проект для глубокого погружения в Python Quart: асинхронный веб‑фреймворк, сохраняющий синтаксис и удобство Flask.

Цель: освоить асинхронное веб‑программирование на Python, используя Quart как «асинхронный Flask».

Ключевые задачи:

понять отличия Quart от Flask в контексте async/await;

научиться строить асинхронные маршруты и обработчики;

освоить работу с асинхронными БД и внешними API в Quart;

сохранить привычную Flask‑стилистику (маршрутизация, шаблоны, контекст запроса).
